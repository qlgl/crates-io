# Registry

``` bash
[registries."qlgl"]
index = "https://github.com/shellcodesniper/crates.io.git"

[registreis."caramella"]
index = "https://it.caramella.kr/crates.io/index.git"

[registry]
default = "qlgl"
```

## Register
https://crates.ql.gl/
-> 회원가입은 막혀있는 상태.

## Login
`cargo login --registry qlgl`

## Package Publishing
`cargo publish --registry qlgl`
